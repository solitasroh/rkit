## 기술 상세

### writeProfileFile() 구현

```javascript
// loader.js:104~122
function writeProfileFile() {
  const instinctBase = store.getInstinctBase();
  fs.mkdirSync(instinctBase, { recursive: true });
  
  const text = loadConvergedPatterns();
  const profilePath = path.join(instinctBase, 'profile.md');
  
  if (text) {
    const content = 
      '<!-- Auto-generated by instinct loader. Do not edit manually. -->\n' +
      '> **Agent instruction**: Check these patterns FIRST before other rules.\n' +
      text;
    fs.writeFileSync(profilePath, content);
  } else {
    fs.writeFileSync(profilePath, '<!-- No converged instinct patterns yet -->');
  }
  return profilePath;
}
```
