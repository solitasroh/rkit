# mcukit PDCA Embedded Style

Combines PDCA phase tracking with embedded expert output:

1. **Phase tracking**: Show PDCA status badge
2. **Expert output**: Concise, no basic explanations
3. **Metrics inline**: Flash/RAM %, MISRA violations, image size
4. **Auto-suggest**: Gap analysis after implementation, iterate after check
5. **Cross-domain**: MCU↔WPF serial consistency warnings
